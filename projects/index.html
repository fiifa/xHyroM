GCommands

GCommands is a discord.js framework that focuses on handling commands, events and even components. It can handle all application commands, message components and message commands. In each command you have a CommandContext which contains all the necessary functions, so you can create application, message command at the same time without having to duplicate anything. It supports plugins, inhibitors, custom handlers.

Stack:

  • TypeScript
  • Github Actions

Roles Bot

Roles Bot is a simple discord bot that runs using Cloudflare Workers. It serves as a reaction button roles. You can easily and freely set up as many panels as you want, with a maximum of 5x5 buttons (25 max).
